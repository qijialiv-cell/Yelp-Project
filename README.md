# Yelp-Dataset

Academic research materials for analyzing the Yelp online review platform.

## File Structure

```
Yelp-Dataset/
├── .DS_Store                    # Mac system file
├── literature_review.html      # Key file: Interactive HTML literature review
├── literature_review.md        # Markdown version of literature review
├── project_context.md          # Project context and guidelines
├── README.md                   # This file
├── wos_export_data.txt         # Web of Science literature records
└── wos_search_query.txt        # Web of Science search query
```

## Key File

**literature_review.html** - Interactive literature review of 35 research papers on Yelp dataset applications.

## Dataset

**Download**: https://business.yelp.com/data/resources/open-dataset/

**Size**: 4.35 GB (compressed) / 8.65 GB (uncompressed)
